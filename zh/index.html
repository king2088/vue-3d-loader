Vue 3d loader

基于threeJS的vue 3D展示组件

快速上手 项目简介

同时加载多个3D模型

可同时加载多个3D模型或加载多个不同类3D模型。

支持vue2/3

vue3请安装2.0.0及以上版本，vue2请安装1.x.x版本。

# 如何安装

# 在你的项目中安装
yarn add vue-3d-loader

# 在你的项目中安装
npm install vue-3d-loader
