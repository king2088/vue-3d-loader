# 事件

## mousedown

鼠标按下事件  
mousedown(mouseEvent, intersects)  
mouseEvent: 鼠标事件  
intersect: 当前相交最近的物体  

## mousemove

鼠标移动事件  
mousemove(mouseEvent, intersects)  
mouseEvent:鼠标事件  
intersect:当前相交最近的物体  

## mouseup

鼠标放开事件  
mouseup(mouseEvent, intersects)
mouseEvent:鼠标事件  
intersect:当前相交最近的物体  

## click

点击事件  
click(mouseEvent, intersects)  
mouseEvent:鼠标事件  
intersect:当前相交最近的物体  

## load

加载模型事件

## process

加载进度事件  
process(event, fileIndex)  
event:加载事件  
fileIndex:当前加载模型的索引  

## error

错误事件  
error(event)  
event:错误事件
