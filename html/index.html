Vue 3d loader

VueJS + ThreeJS 3d viewer component

Get started Introduce

Load multiple 3D models simultaneously

Multiple 3D models can be loaded at the same time or multiple 3D models of different classes can be loaded.

Support vue2/3

Vue3 please install 2.0.0 or later, vue2 please install 1.x.x version

# How to install?

# yarn
yarn add vue-3d-loader

# npm
npm install vue-3d-loader
