# Loader draco model

TIP

Need to download Draco repository storage with local static folder of your project, download url: Draco decoderopen in new window，Unzip it and place it in the assets directory of your local project. The draco path should be："assets/draco/gltf/"

<template>
  <vue3dLoader
    filePath="/models/gltf/LittlestTokyo.glb"
    :cameraPosition="{ x: 10, y: 700, z: 1000 }"
    :enableDraco="true"
    outputEncoding="sRGB"
    :height="350"
    backgroundColor="#F2F2F2"
    dracoDir="../../assets/draco/gltf/"
  />
</template>
<script lang="ts" setup>
import { vue3dLoader } from "vue-3d-loader";
</script>
